# Index And Log Rules

- `index.md` is the routing layer for the wiki; every page in `pages/` should be listed there.
- `log.md` is append-only; add new entries instead of rewriting history.
- Keep `README.md` and `CLAUDE.md` aligned with the actual repo structure.
- If paths or directories change, update all top-level docs in the same slice.
